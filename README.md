# Sales Analysis & Prediction — Allied Company 📈

End-to-end commercial analytics project for a cleaning 
products company with national presence in Mexico.

## Objective
Analyze sales performance by category, region and segment,
and generate forecasts to support strategic decisions.

## Tools
- Python (pandas, matplotlib, statsmodels)
- SQL
- AWS (S3, Athena)
- Excel
- Looker Studio

## Process
1. Initial exploration with Excel and pivot tables
2. Relational modeling in SQL with joins
3. Data cleaning and EDA with Python
4. Sales forecasting with ARIMA model
5. Cloud analysis with AWS S3 + Athena
6. Interactive dashboard in Looker Studio

## Results
- MAE optimized Vanish: 65.72
- MAE optimized Lysol: 0.47
- Area 2 highest sales volume (10.77% of total)
